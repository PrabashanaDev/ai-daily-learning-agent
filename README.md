# 🤖 Auto-Learning Agent
An autonomous Python agent that generates a daily programming concept using Google Gemini and logs it automatically via GitHub Actions.

## How to Install and Use This Tool
1. Click the green **Use this template** button at the top right of this page to copy this repository to your account.
2. Go to **Google AI Studio** and get a free Gemini API Key.
3. In your new repository, go to **Settings > Secrets and variables > Actions**.
4. Click **New repository secret**, name it `GEMINI_API_KEY`, and paste your key.
5. Go to the **Actions** tab and enable workflows. 
The agent will now wake up every day at 10:00 AM, write a new tech note, and commit it to your repository!
